================================================================================
온노드 실시간 클라우드 연동 데모 (v5.0) — 시연 가이드
================================================================================

개요
----
복잡한 설치형 프로그램이나 파이썬 코드가 아니라, 크롬 브라우저만 있으면 전 세계
어디서든 스마트폰과 노트북이 실시간으로 연동되는 '웹 기반 실시간 시연 시스템'입니다.

작업 방식
---------
• 크롬 브라우저 연결: Firebase(구글 클라우드 DB)를 활용해 별도의 서버 구축 없이도
  노트북에서 띄운 화면을 스마트폰에서 조작하면 데이터가 빠르게 동기화되도록 설계했습니다.

• Cursor 등 에디터: 프로젝트를 열고 의존성 설치 후 실행하면 동작합니다.
  (Firebase 프로젝트 생성 후 .env에 VITE_FIREBASE_CONFIG 설정 필요)

시연 방법 (현장 가이드)
-----------------------
이 데모는 실제로 구글 클라우드(Firebase)와 통신하는 시스템입니다.

1. 노트북 준비
   - 노트북에서 앱을 실행한 뒤(예: npm run dev 후 브라우저에서 열기),
     [노트북(대시보드)] 버튼을 클릭합니다.
   - 고유한 세션 ID(예: K3B8Z)가 생성됩니다.

2. 스마트폰 접속
   - 노트북 대시보드에서 [URL 복사하기]로 복사한 주소를
     카톡·메일 등으로 본인 스마트폰에 보낸 뒤, 스마트폰의 크롬 브라우저에서 엽니다.
   - 스캔 없이 주소만으로도 ?session=세션ID 가 포함되어 즉시 같은 세션에 연결됩니다.
   - (참고: 원문에 언급된 외부 단축 URL은 실제 배포 URL로 대체해 사용하세요.)

3. 실시간 연동 보여주기
   • 노트북 화면을 그대로 두고, 스마트폰에서 'ON-VOCAL' 또는 'ON-KOREAN' 탭을 눌러
     노트북 화면의 탭(타입)이 실시간으로 바뀌는 것을 보여줍니다.
   • 스마트폰에서 [참여 완료 & 분석 전송] 버튼을 누릅니다.
   • 심사위원에게 노트북 화면을 가리키며
     "지금 분석 중(Analyzing)이라는 메시지가 뜨며 클라우드 분석이 진행되고 있습니다"
     라고 설명합니다.
   • 약 1.5초 후 수치가 올라가고 리포트가 업데이트되는 것을 보여주면 시연을 마칩니다.

전략적 메시지 (초대장 모달)
----------------------------
• 랜덤 플레이 댄스 타겟팅: "홍보 비용 대신 랜덤 플레이 댄스 채널 및 기술 파트너십으로
  자발적 노출" — 자생적 마케팅 논리는 심사에서 설득력 있는 스토리로 활용할 수 있습니다.

• 대시보드 [공식 초대장 발송 시연] 버튼으로 VIP Invitation 모달을 띄워
  인플루언서 파트너십 메시지를 함께 전달할 수 있습니다.

Firebase 콘솔 참고
------------------
• Authentication: 익명 로그인(Anonymous) 사용을 켜야 합니다.
• Firestore: artifacts/{appId}/public/data/sessions/{sessionId} 경로에
  읽기/쓰기가 가능하도록 규칙을 설정해야 합니다. (시연용은 보안 규칙을 엄격히 검토하세요.)
